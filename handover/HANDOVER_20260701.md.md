## 2. 案B改修の実装詳細

### 共通の設計思想

**「数値上の間隔」と「視覚的な間隔」の乖離**を解消するため、以下の3点セットで対応：
1. ラベル箱を帯本体より上下2pxずつ縮める（立体感演出）
2. ラベル⇔本文間隔を40pxに拡大
3. ラベル箱の位置を`+LABEL_MARGIN`シフト

### 具体的な数値

| 帯 | 帯本体高さ | ラベル箱高さ | 間隔 |
|---|---:|---:|---:|
| C-1 結論帯 | 38px | **34px**（-2px上下） | **40px** |
| C-2 POINT帯 | 32px | **28px**（-2px上下） | **40px** |
| C-3 改善方向帯 | 36px | **32px**（-2px上下） | **40px** |

### コード変更例（C-1結論帯）

```python
CONCL_TOP = 614
CONCL_H = 38
LABEL_MARGIN = 2
LABEL_H = CONCL_H - LABEL_MARGIN * 2  # 34px
LABEL_W = 88
GAP = 40

# 本体
add_shape(slide, MSO_SHAPE.RECTANGLE, 40, CONCL_TOP, 1200, CONCL_H,
          fill=WHITE, line=ORANGE, line_width_pt=1)
add_shape(slide, MSO_SHAPE.RECTANGLE, 40, CONCL_TOP, 1200, CONCL_H,
          fill=ORANGE, fill_alpha=10000)
# ラベル箱（上下2pxマージン付き）
add_shape(slide, MSO_SHAPE.RECTANGLE,
          40, CONCL_TOP + LABEL_MARGIN, LABEL_W, LABEL_H, fill=ORANGE)
# 本文（40px間隔）
body_left = 40 + LABEL_W + GAP  # = 168
body_w = 40 + 1200 - body_left  # = 1072
```
