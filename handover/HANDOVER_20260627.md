## 6. 重要な技術メモ

### Project と Computer の同期問題
- Project（永続）とComputer（一時）が乖離することがある
- Edit中に「String not found」エラーが出たら、まず `save_computer_file_to_project` で同期し直す
- 大きな改修時は `python3` スクリプトで一括置換が安全（Editのstring matchより堅牢）

### Instructions の文字数制限
- ChatGPT GPTs Instructions欄は **8,000文字制限**
- 現状の `01_Instructions_WebDiagnosis_lite.md`：7,817文字（残り183文字）
- フェーズ1.5でデザインシステム言及を追加する場合、**文字数調整が必要になる可能性**
- 必要なら design_system.md への参照リンクで節約

### Code Interpreter のタイムアウト
- 4ファイル同時生成（個別3 + 統合1）は2分超で危険
- Q18=[B] により「統合版は明示要求時のみ」で対応
- タイムアウト時はC-1/C-2のみで確実納品（J-4フォールバック）
