### C-3 ビジュアル診断ボード

| 項目 | 上限 |
|---|---:|
| summary（総評） | 80字 |
| top_issue | 40字 |
| direction | 50字 |
| sections[].label | 20字 |
| flow_steps[]（label+note合計） | 25字 |
| highlights[].title | 30字 |
| highlights[].before / .after | 各40字 |

### バリデーション失敗時の挙動

```
[文字数バリデーションエラー]
  項目: C-2 提案#1 POINT (proposals[0].point)
  上限: 150文字（design_system.md §1 準拠）
  実際: 151文字（1文字オーバー）
  内容: 「...」
→ GPTs側で規定文字数内に要約し直してください。
  ※末尾「…」での省略は禁止です。要点を絞り込んでください。
```

**ValueErrorで停止 → GPTsが要約し直して再実行**。末尾「…」省略は禁止。
